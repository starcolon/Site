Software engineering job hunting

JOB HUNTING?

บล็อกนี้จะบอกเล่าเรื่องราวของ Software engineer (ผมเอง) ที่ออกตามหางานใหม่ในสายซอฟท์แวร์เหมือนเดิม แต่มองหางานที่ใช้ทักษะทาง engineering มากขึ้น (ใช้ความรู้ algorithm หนัก ๆ หรือถ้าใช้ machine learning ด้วยนี่จะเยี่ยมไปเลย) เพราะงานทำ business software ง่าย ๆ แบบเดิมมันเริ่มไม่สนุกแล้ว เมื่องานประจำไม่สนุก ก็ต้องออกรบในสนามรบใหม่ ก่อนออกรบ ผมก็ตรวจสอบอาวุธหนักเบาของตัวเองซะก่อนครับ

✌ อาวุธก่อนออกรบของผม ✌

บวกเพิ่มอีกข้อก็คือ ผมสื่อสารภาษาอังกฤษปากเปล่าได้คล่อง สบายมาก จุดนี้ผมจะดันเป็นจุดขายสร้างความแตกต่างจาก candidates คนอื่น ๆ ด้วย

☠ จุดอ่อนของผม ☠

  • ผมคิดช้า ไหวพริบในการตอบโต้สด ๆ ไม่ค่อยดี
  • ถ้าพูดไว ๆ ติดอ่างประจำครับ 55
  • ผมเล่าเรื่องไม่เก่ง ถ้าเล่าเรื่องที่ไม่ได้เตรียมมาก่อน โอกาสฟังไม่รู้เรื่องสูงมากกกก (โตจนป่านนี้ ก็ยังเป็นไม่หาย)

เวลาไปสัมภาษณ์ ผมจะขายจุดเด่นของผมเป็นหลักครับ ส่วนจุดด้อยผมจะพยายามกลบมันด้วยจุดเด่นนั่นแหละครับ (แต่เอาเข้าจริง จุดด้อยก็มาเล่นงานผมตลอดแหละครับ 55)

ทางเลือกงาน Software Engineer ในกรุงเทพ มีอะไรมั่ง?

ผมรู้ตัวดีว่าชอบงานโปรแกรมมิ่งที่ใช้ algorithm หนัก ๆ มากกว่าทำแค่ business software ทั่วไป ก็เลยหางานจากเกณฑ์ตามนี้ครับ

  • งานเน้นทาง Research หรือประมวลผลข้อมูล (จริง ๆ อยากไปสาย quatitative analysis ครับ)
  • ค่าตอบแทนสมน้ำสมเนื้อ benefits ดี
  • เวลาเข้างานไม่เช้าเกินไป เพราะผมนอนดึกตื่นสาย (-_-)
  • ถ้าได้ใช้ภาษาอังกฤษสื่อสารปากเปล่าบ่อย ๆ จะดีมาก

Potential jobs ที่ผมเล็งไว้ที่น่าสนใจ ก็มีตามนี้ครับ

  • JavaScript Engineer ที่ Thomson Reuters
  • Software Engineer (Voice recognition) ที่ AmiVoice
  • Game Developer (C++ on Mobile) ที่ Boomzap Entertainment (Singapore)
  • Big Data Engineer ที่ Agoda
  • Software Engineer (ASP.NET/AWS) ที่ Jetabroad
  • Quantitative Developer ที่ WorldQuant
  • Software Engineer (C++) ที่ Bombadier

ตัวเลือกที่น่าสนใจทั้งหมดข้างต้นนี่ผมประเมินตอนแรกว่าน่าจะสนุกและได้ใช้ algorithm จริงจัง ไม่น่าจะเป็นงานทำ business app ทั่วไปที่น่าเบื่อแน่ ๆ ว่าแล้วก็โดนเข้าสู่กระบวนการสัมภาษณ์ครัช

สัมภาษณ์ที่ AMIVOICE

ทำไมเลือกสมัครที่นี่?

Amivoice เป็นบริษัททำ Voice recognition ของญี่ปุ่น พูดง่าย ๆ คือออกแบบระบบแปลงเสียงพูดให้เป็นตัวอักษรโดยไม่ต้องใช้คนครับ งานที่นี่เป็นกึ่ง develop กับ research ได้ใช้ความสามารถทางด้าน machine learning และ algorithm หนัก ๆ แน่นอน งานคงไม่มีคำว่าน่าเบื่อ ไร้สาระแน่ ๆ ครับ :)

ติดต่อนัดสัมภาษณ์

ผมสมัครผ่าน head hunter ครับ ได้รับการติดต่อเข้าไปสัมภาษณ์และทดสอบความสามารถหลังจากที่ส่ง CV เข้าไปเกือบ 2 สัปดาห์

กระบวนการสัมภาษณ์และทดสอบความสามารถ

AmiVoice ทดสอบผู้สมัครด้วยข้อสอบ programming วัดความรู้พื้นฐาน และความรู้ด้าน algorithm เบื้องต้น ไม่มีอะไรยากครับ และวัดความสามารถ SQL ซึ่งก็ไม่ยากเช่นกัน โดยรวมเป็นการทำข้อสอบที่สนุกมาก โดยไม่ได้บังคับว่าคุณจะเขียนอัลกอริทึมด้วยภาษาอะไร คุณสามารถเลือกภาษาอะไรก็ได้ที่ถนัด งัดออกมาเขียนแก้ปัญหาจากโจทย์

First impression?

Project manager และ MD ของ AmiVoice มีความสุภาพสูงมาก ให้เกียรติผู้สัมภาษณ์และมีบรรยากาศการสัมภาษณ์ที่สนุกสนาน ผมชอบมาก ๆ ไม่มีอะไรเครียด ทำให้สามารถพูดคุยและถามตอบได้ต่อเนื่อง (ยาวนาน 55+) แต่ที่น่าสังเกตคือการพูดคุยเล่าประสบการณ์การทำงาน และความสามารถ ทาง MD สนใจทุกรายละเอียด แต่ไม่ได้สนใจงานโปรเจกต์ส่วนตัวแต่อย่างใดครับ เขาพิจารณาประสบการณ์ในองค์กร และโปรเจกต์ตอนเรียนเท่านั้นเอง

ผลการสัมภาษณ์?

☑ ผมได้รับ offer จาก AmiVoice (ดีใจมาก) ซึ่งผมประทับใจมาก เพราะ AmiVoice ใช้ Ruby / Python / Asterisk / Java / C++ ในการพัฒนาผลิตภัณฑ์ (งานทำบน Mac OSX ครับ) แต่ถ้าย้อนขึ้นไปดู skill ผมข้างบนภาษาพวกนี้ไม่ใช่จุดขายผมเลย ไม่ได้ใส่ใน CV ด้วยซ้ำ แต่สกิลด้าน อัลกอริทึมและงาน research ที่ผ่านมาของผมสร้างความเชื่อมั่นให้กับ AmiVoice ทำให้ได้รับ offer ที่ค่าตัวสมน้ำสมเนื้อ

เป็นไปตามความคาดหวังหรือไม่?

ไม่มีอะไรผิดไปจากความคาดหวัง จากการฟัง Project manager และ MD บอกเล่าตัวผลิตภัณฑ์และลักษณะงาน ผมรู้สึกว่างานเจ๋งดี น่าสนใจมาก

THOMSON REUTERS

ทำไมเลือกสมัครที่นี่?

ผมเชื่อว่า Software engineer ไทยทุกคนทราบดีว่ารอยเตอร์สมีสวัสดิการที่ดีเยี่ยม บริษัทมีคุณภาพ คนข้างในก็มีคุณภาพ ถ้าได้เข้าไปร่วมงานด้วยสนุกแน่นอนครับ

ติดต่อนัดสัมภาษณ์

ผมได้รับการติดต่อจาก HR หลังส่ง CV ไปเพียง 2 วัน รวดเร็วทันใจมาก :) โดยได้มาทราบภายหลังว่าผมโชคดีเป็นหนูลองยาคนแรกเลยในการสัมภาษณ์หนนี้ 55

กระบวนการสัมภาษณ์และทดสอบความสามารถ

ตามสไตล์ของรอยเตอร์ส ผู้สมัครจะต้องทำข้อสอบข้อเขียนเพื่อทดสอบความสามารถ programming เสียก่อน (เหมือนข้อสอบในมหาวิทยาลัยนั่นแหละครับ ยากง่ายพอ ๆ กัน)

จากนั้นในขั้นตอนการสัมภาษณ์ manager ที่มีความรู้ technical ที่สูงมากก็จะรัวยิงคำถามใส่ผมเพื่อทดสอบความสามารถและไหวพริบในการตอบคำถาม บอกเลยว่าคำถามมีทั้งยากและงาย บางอันง่ายแต่ดันตอบผิด 555+ คำถามเกือบทั้งหมดเป็นความรู้ technical ครับ

ระหว่างการสัมภาษณ์ที่ยาวนาน จะมีการให้โจทย์เขียนโค้ดสด ๆ บนกระดาน โดยให้เวลาคิด 15 นาที โจทย์ไม่ยากเลยครับ ทักษะลอจิกและคณิตศาสตร์ธรรมดา แต่ผมทำผิด 555 จะแก้ก็ไม่ทันแล้ว ตอนนั้นตื่นเต้นด้วยกลัวพลาด (ก็เลยเกร็งจนพลาดจริง ๆ)

First impression?

5 คนรุมสัมภาษณ์ครับ ตกใจสิ *_*'' // ที่น่าประทับใจมากคือ HR อำนวยความสะดวกตลอดกระบวนการให้ดีมาก ๆ

ผลการสัมภาษณ์?

☑ ผมได้รับ offer หลังผ่านพ้นการสัมภาษณ์ไปกว่า 2 สัปดาห์ (นานจนลืม 555) ซึ่ง HR แจ้งแต่แรกและแจ้งตลอดว่ามีผู้สัมภาษณ์ตำแหน่งนี้เยอะมาก (กว่าจะสัมภาษณ์ครบหมด ปาไป 2 สัปดาห์) เลยทำให้ผลการคัดเลือกออกช้าครับ

เป็นไปตามความคาดหวังหรือไม่?

ราคาค่าเหนื่อยที่รอยเตอร์สเสนอมานั้นต่ำกว่าที่คาดหวังไว้พอสมควร แต่หากไม่นับเรื่องค่าเหนื่อย ผมคิดว่าสวัสดิการอื่น ๆ ของรอยเตอร์สอยู่ในระดับที่โหดมาก เพียงแต่จากการได้ไปคุยรายละเอียดตัวงานแล้ว น่าสนใจครับ แต่ยังไม่น่าสนใจมากเท่างานกึ่ง research ที่ AmiVoice

JETABROAD

ทำไมเลือกสมัครที่นี่?

Jetabroad เป็นบริษัทสัญชาติออสเตรเลียที่ทำระบบจองตั๋วเครื่องบิน มีหน้าบ้านที่สวยงามใช้งานง่าย แถมมีบรรยากาศการทำงานสมัยใหม่ ได้ร่วมทีมกับโปรแกรมเมอร์เทพ ๆ จากออสเตรเลียและไทย

ติดต่อนัดสัมภาษณ์

CTO ของ Jetabroad ติดต่อผมในวันรุ่งขึ้นหลังจากสมัครผ่าน e27 ซึ่งไวจนผมตกใจ

กระบวนการสัมภาษณ์และทดสอบความสามารถ

สัมภาษณ์โดยตรงกับ CTO และโปรแกรมเมอร์เทพชาวออสเตรเลีย และเทพชาวไทยครับ เป็นการสัมภาษณ์ที่สนุกและเป็นกันเองมาก ผู้สัมภาษณ์ใส่ใจรายละเอียดของโปรไฟล์ผมมาก ลองเล่นเกมของผมมาก่อนอีกตะหาก 555+ การสัมภาษณ์เป็นการถามความรู้ทาง technical ล้วน ๆ เหมือนรอยเตอร์ส

หลังกระบวนการสัมภาษณ์ ผมได้รับโจทย์ให้มาทำโปรเจกต์เล็ก ๆ โดยให้เวลา 1 สัปดาห์ โจทย์ไม่ยากเกินไปแต่ก็ไม่ง่ายเพียงพอที่จะทำให้เสร็จในคืนเดียว ผมใช้เวลา 2 คืนในการทำ ถือว่าเป็นโจทย์โปรเจกต์ที่สนุกครับ เวลาที่เสียไปในการทำโปรเจกต์น้ถือว่าคุ้มค่ามากครับ เพราะได้ความรู้หลายอย่างเลย :)

First impression?

คนที่ Jetabroad เป็นกันเองมาก เป็นการพูดคุยที่ประทับใจมาก บรรยากาศออฟฟิศของที่นี่ดีมากทีเดียว วิวกระจกรอบด้านเห็นวิว All seasons place ได้สบาย บรรยากาศแบบ startups ครับ

ผลการสัมภาษณ์?

☑ ผมได้รับ offer ที่เยี่ยมยอดจาก Jetabroad ภายในไม่กี่วันหลังจากที่ผมส่งโปรเจกต์ครับ เป็นการตอบรับที่รวดเร็วตามคาด ^ ^

เป็นไปตามความคาดหวังหรือไม่?

Jetabroad ดีเกินความคาดหมายผมมาก ระบบการทำงานใช้ Kanban ผสม Scrum แต่บางครั้งถ้าโปรเจกต์ขนาดใหญ่จะใช้ Agile (แต่ไม่บ่อย) บรรยากาศการทำงานผมให้ 10/10 เลยครับ อ้อ ที่นี่ให้จอ Cinema display ขนาดเท่าโต๊ะกินข้าวในการทำงานเลยนะครับ สวดยวดมาก

BOOMZAP ENTERTAINMENT

ทำไมเลือกสมัครที่นี่?

Boomzap เป็นบริษัทเกมบนหลายแพลตฟอร์ม ผลงานเยอะมากครับ ความสุดยอดของที่นี่คือโปรแกรมเมอร์ทำงานจากที่ไหนก็ได้ ไม่มีออฟฟิศครับ ติดต่องานกัน ส่งงานกันผ่านอินเตอร์เน็ตล้วน ๆ (ถ้ามีเวลาว่างเล่นเน็ต วันละ 2-3 ชั่วโมงล่ะก็ นี่แหละงานผ่านเน็ต #ผิด)

ติดต่อนัดสัมภาษณ์

CEO ของ Boomzap เป็นคนดูแลการหาคนร่วมงานเอง และติดต่อผมในวันรุ่งขึ้นหลังจากส่ง CV สมัครไป รวดเร็วฉับไวมาก

กระบวนการสัมภาษณ์และทดสอบความสามารถ

Boomzap ไม่มีออฟฟิศ แล้วจะสัมภาษณ์อย่างไร? ไม่ใช่ปัญหา ขั้นตอนด่านแรกคือ CEO จะส่งชุดคำถาม (หลายข้อมากๆ) มาทางเมล ให้ส่งคำตอบไป คำถามนั้นผสม ๆ กันระหว่างคำถามด้าน technical และทัศนคติต่อการเป็น game developer ครับ โดยรวมถือว่าตอบยากมาก

First impression?

จุดน่าสนใจหลักๆ เลยคือเป็นบริษัทเกมที่ผลงานดี ๆ เยอะมาก (เกมแนว casual บนโมบายล์) และสามารถทำงานจากที่ไหนก็ได้ อันนี้เด็ดมาก

ผลการสัมภาษณ์?

หลังส่งคำตอบของชุดคำถามที่ CEO ส่งมาให้ในตอนแรก ก็จัดการแอด Skype กันไปเพื่อรอสัมภาษณ์​แต่แล้วก็ไม่ได้สัมภาษณ์ครับ (อ้าว 555+) กระบวนการยุติไปแบบงง ๆ

เป็นไปตามความคาดหวังหรือไม่?

ไม่ทันจะได้สัมภาษณ์ต่อครับ ไม่รู้จะคาดหวังอะไร T_T

แล้ว AGODA / WORLDQUANT / BOMBADIER ล่ะ?

Agoda มีกระบวนการคัดเลือก candidate ที่ยาวนานมากครับ ผมเพิ่งได้รับการติดต่อสัมภาษณ์ทางโทรศัพท์หลังส่ง CV เข้าไปถึงสองสัปดาห์ และ HR แจ้งว่าจะนำเข้ากระบวนการคัดสรรต่ออีกสองสปัดาห์ = ='' นานขนาดนี้ผมไม่สามารถรอได้แล้วครับ

ส่วน Worldquant กับ Bombadier นั้นน่าสนใจมากทีเดียว น่าเสียดายมากที่โปรไฟล์ผมไม่สามารถสร้างความน่าสนใจได้ ไม่ได้รับการเรียกสัมภาษณ์แต่อย่างใดครับ (โฮ T_T)

Final Decision!

Offer ที่ได้มาแต่ละที่ผมมองว่าระดับดีเยี่ยมทั้งนั้นครับ เลือกยากมาก แต่ละที่ก็มีลักษณะงานที่ต่างกันสิ้นเชิง ผมเลยลองเปรียบเทียบดูตามนี้ครับ
