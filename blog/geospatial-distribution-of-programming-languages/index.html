Analysis of Geo-Distributions of Programming Languages
7 Aug 2016 | Tao P.R.

A quick hypothesis of this analysis project is just simple. We believe the popularity of programming languages are likely epidemic. It might be controversial if world-famous languages like C++, Python, Java, etc. are dominating everywhere in the world. We rush into finding out!

Languages of Github Repos

Languages are a common attribute of a Github repository. Github API does a great job on segmenting all programming languages which co-exist in a particular repository with given amount of code and these values are accessible out-of-the-box through its API. The endpoint users (us) don't need to commit any additional effort to find out.

Download Github Data

We prepared a quick python script to download Github repo data and store them in MongoDB. Due to the API usage constraints, we are unable to download 90K repos at once even our Internet connection is ready to go. We accumulatively collect the data by days until it reaches 90K reposotories (as a sample set).

Determine the Location?

We determine the location of code written in a repository by look up the location of the owner. This is a bit of tricky though since some people may don't properly have valid address attributed. To convert a plain text address to a geospatial location, we use GoogleMap geocoding API. See the implementation here.

Geospatial Plots!

It's a show time! Once we have enough Github repo data downloaded (say 90K repos), we proceed to plot it with GoogleMap Heatmap API. The API is not fully free for server mode and we can't afford the price (apology), so we run it offline and capture the visualisations in the following showcase.
