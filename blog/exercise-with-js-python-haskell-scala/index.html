EX1: Generate a list of tuples of counting numbers (1,2),(3,4),(5,6),...

JS

JavaScript is very handy at this job. Go for ES6 generator for lazy evaluation. 
					
						function* gen(n){
							yield [n,n+1];
							yield* gen(n+2);
						}

						var tuples = gen(1); // Initiate
						// Go...
						tuples.next().value; // [1,2]
						tuples.next().value; // [3,4]
						tuples.next().value; // [5,6]

Python

Let's make an infinite generator for tuple in a similar way.
				
					def gen(n):
						while True:
							yield (n,n+1)
							n = n + 1

					# Initiate
					tuples = gen(1)

					# Go go!
					next(tuples) # (1,2)
					next(tuples) # (3,4)
					next(tuples) # (5,6)

Haskell

People say dammit. It's too easy. 
				
					-- Infinite run
					[(i,i+1) | i <- [1,3..]] 

					-- [(1,2),(3,4),(5,6).......... until you beg it to stop.

EX2: Fibonacci tuples (1,1),(2,3),(5,8),...

JS

Generator comes into play yet again. 
					
						function* gen(a,b){
							yield [a, b];
							yield* gen(a+b, a+2*b);
						}

						var tuples = gen(1,1); // Initiate first 2 elements
						// Go...
						tuples.next().value; // [1,1]
						tuples.next().value; // [2,3]
						tuples.next().value; // [5,8]

Python

Similar formula with the preceding JS snippet. 
				
					def gen(a,b):
						while True:
							yield (a,b)
							a, b = a+b, a+2*b

					# Initiate
					tuples = gen(1,1)

					# Go go!
					next(tuples) # (1,1)
					next(tuples) # (2,3)
					next(tuples) # (5,8)

EX3: Given a list of triplets (a,b,c), find ones which form right triangles
and return areas of triangles they form.

NOTE: This is literally so called Pythgorean Triple

JS

Filter and Map can help. 
					
						var candidates = [[1,1,5],[3,4,5],[5,12,13],[10,20,30]];
						function rightTriangle(a,b,c){
							return c*c == a*a + b*b
						}
						function area(a,b,c){
							return 0.5*a*b
						}

						candidates.filter(rightTriangle).map(area);

						// Filtered ..
						// [[3,4,5],[5,12,13]

						// Mapped ..
						// [6.0,30.0]

Python

Filter and Map can help. 
					
						candidates = [[1,1,5],[3,4,5],[5,12,13],[10,20,30]]
						def rightTriangle(a,b,c):
							return c**2 == a**2 + b**2

						def area(a,b,c):
							return 0.5*a*b

						[area(a,b,c) for a,b,c in candidates if rightTriangle(a,b,c)]

						# [6.0,30.0]

Haskell

Likewise, writing a chain of operations in Haskell is a real joy. 
					
						let candidates = [(1,1,5),(3,4,5),(5,12,13),(10,20,30)]

						-- Pythagorean triple checker
						let right_triangle (a,b,c) = c*c == a*a + b*b

						-- Area
						let area (a,b,c) = 0.5 * a * b

						-- Do the pipeline
						map area $ filter right_triangle candidates

						-- [6.0, 30.0]

EX4: Circular shift. Right shifting [1,2,3,4] by 3 is supposed to result in [2,3,4,1]

JS

Let's try it with recursion. 
					
						function shiftR(arr,n){
							var list = arr.slice();
							var a = list.pop();
							list.splice(0,0,a);
							(--n > 0) && (list = shiftR(list,n));
							return list;
						}

						// Try it
						shiftR([1,2,3,4],3); // [2,3,4,1]
						shiftR([1,2,3,4,5,6,7],8); // [7,1,2,3,4,5,6]

Python

Explicitly translate the previous algorithm to Python. 
					
						def shiftR(arr,n):
							list = arr[:]
							a = list.pop()
							list = [a] + list
							return shiftR(list,n-1) if n>0 else list

						# Try
						shiftR([1,2,3,4],3) # [2,3,4,1]
						shiftR([1,2,3,4,5,6,7],8) # [7,1,2,3,4,5,6,7]

Haskell

This can be done recursively within a one-liner guarding. 
					
						-- Recursion comes into play
						let shiftR xs n | n==0 = xs | otherwise = shiftR ([last xs] ++ init xs) n-1

						-- Try it on
						shiftR [1,2,3,4] 3 
						-- [2,3,4,1]

						shiftR [1,2,3,4,5,6,7] 8
						-- [7,1,2,3,4,5,6,7]

EX5: Proper divisors of an integer which are also prime numbers.

JS

Implementing this in an exhaustive way would involve some loops. 
					
						function isDivisible(a){
							return function(b){
								return a>b && a%b==0
							}
						}

						function properDivisor(n){
							var divisors = [];
							for (var a=2; a<n; a++){
								if (isDivisible(n)(a) && !divisors.some(isDivisible(a))){
									divisors.push(a);
								}
							}
							return [1].concat(divisors)
						}

						// Now try
						properDivisor(18); // [1,2,3]
						properDivisor(188); // [1,2,47]
						properDivisor(1888); // [1,2,59]
						properDivisor(2000); // [1,2,5]
