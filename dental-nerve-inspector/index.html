What is this project?

The mission of the project is to create an automated system which has an ability to detect and locate the nerve line accurately from the sequence of the CT scanned images. That detected curve from the image is then utilized as a guide line for a panorama projector which renders a sequence of CT scanned images into a linear plane.

How it works?

The algorithm is designed to roughly detect the nerve line in the image. The roughly estimated curve is then iteratively refined until it represents the curvature of the detectable nerve curve in the image with the least errors.
The detected curved is aimed to be used as a panorama pivots so it needs to be smooth so the rendered panorama image is helpful rather than looks dizzy.

Slightly more technical?

Since it makes sense to say most of the CT scanned images represent various kinds of the visible defects of teeth, the algorithm so needs to be flexible enough to overcome those obstacles.
The very common defects are lost teeth, appearance of artificial teeth, fillings. Those defects easily mislead the naive algorithms because any holes, displacements of the teeth could confuse the detection. However there is no excuse the algorithm must function properly and accurately despite an occurrence of those defects.
